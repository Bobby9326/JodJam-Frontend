# JodJam — Frontend

> จดจำ · หนึ่งวัน หนึ่งความทรงจำ

**JodJam** is a daily memory journal — one entry per day, no backdating, memory is final.

🌐 **Live** → [jodjam.vercel.app](https://jodjam.vercel.app)  
🔧 **Backend** → [github.com/Bobby9326/JodJam-Backend](https://github.com/Bobby9326/JodJam-Backend)

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + Vite |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Data Fetching | TanStack Query |
| Global State | Zustand |
| HTTP Client | Axios |
| Date Utility | Day.js |
| Validation | Zod |
| Charts | Recharts |
| Deploy | Vercel |

## Pages

| Route | Page | Description |
|---|---|---|
| `/login` | Login | Google OAuth sign in |
| `/` | Calendar | Yearly memory calendar |
| `/memory/create` | Create Memory | Record today's memory |
| `/memory/:date` | Memory Detail | View memory by date |
| `/profile` | Profile | User profile & stats |
| `/stats` | Statistics | Mood & rating analytics |

## Local Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Setup

```bash
# Clone
git clone https://github.com/Bobby9326/JodJam-Frontend.git
cd JodJam-Frontend

# Install dependencies
npm install

# Copy environment file
cp .env.example .env
# Fill in your values in .env

# Run
npm run dev
```

App will be available at `http://localhost:5173`

## Key Design Decisions

- **Cookie-based auth** — `httponly` cookies managed by backend, frontend uses `credentials: include` on every request
- **Auto refresh** — Axios interceptor catches 401 and silently refreshes token
- **Immutable memories** — No edit or delete UI, enforced by backend
- **Private storage** — All images served via signed URLs generated by backend
